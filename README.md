### Step 3 - 전처리
```
✔ 필요한 컬럼만 추출 (10개 독립변수 + 범죄율)
✔ 결측치 처리
✔ 코로나 기간 이상치 처리 (2020~2021)
✔ 외국인 비율 계산 (등록외국인수 ÷ 인구수 × 100)
✔ 도시명 더미변수 처리 (패널 데이터 대응)
✔ 데이터 표준화 (StandardScaler)
```

#### ⚠️ 보완 1: 단위(Scale) 불일치 → 표준화 필수
인구 수, GRDP, 기온, 조이혼율 등 단위가 완전히 달라서
그대로 회귀분석 돌리면 수치 큰 변수가 과대평가됨

```python
from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)  # 독립변수 표준화
```

#### ⚠️ 보완 2: 패널 데이터 → 도시 더미변수 추가
6개 도시 × 10년 = 패널 데이터
도시 고유 특성을 무시하면 회귀 결과 왜곡 가능

```python
# 도시명을 0/1 컬럼으로 변환
df = pd.get_dummies(df, columns=['지역'], drop_first=True)
# 결과: 도시_대구, 도시_인천, 도시_광주, 도시_대전, 도시_울산 컬럼 생성
```

### Step 4 - 탐색적 데이터 분석 (EDA)
```
✔ 연도별 범죄율 추이 시각화
✔ 도시별 범죄율 비교
✔ 각 변수의 분포 확인
```

### Step 5 - 상관분석
```
✔ 각 독립변수와 범죄율의 상관계수 계산
✔ 히트맵 시각화
✔ 가장 관련 높은 변수 1차 선별
✔ 다중공선성 우려 변수 사전 파악
```

#### ⚠️ 보완 3: 다중공선성(Multicollinearity) 체크
독립변수끼리 너무 강한 상관관계가 있으면 회귀 결과 왜곡됨

```
우려되는 조합:
- 인구 수 ↔ 기초생활수급자수  (인구 많으면 수급자도 많음)
- 지역 소득(GRDP) ↔ 실업률    (소득 높으면 실업률 낮음)
- 지역 소득(GRDP) ↔ 기초생활수급자수
```

### Step 6 - 회귀분석
```
✔ VIF(분산팽창요인) 계산 → 다중공선성 최종 확인
✔ VIF 10 이상 변수 제거 또는 조정
✔ 다중 선형회귀분석 (표준화된 데이터 사용)
✔ 가장 영향력 큰 변수 확인
✔ 모델 성능 평가 (R², p-value)
```

#### ⚠️ VIF 계산 코드
```python
from statsmodels.stats.outliers_influence import variance_inflation_factor

vif_data = pd.DataFrame()
vif_data['변수'] = X.columns
vif_data['VIF'] = [variance_inflation_factor(X.values, i)
                   for i in range(len(X.columns))]

# VIF >= 10 이면 다중공선성 문제 → 해당 변수 제거 검토
print(vif_data.sort_values('VIF', ascending=False))
```

## ⚠️ 주의사항

### 코로나 처리 방침
- 2020 ~ 2021년 데이터는 코로나로 인한 이상치 가능성 있음
- 이상치로 처리하지 않고 **분석에 포함** (10년치 데이터 신뢰도 유지)
- 단, 코로나 영향 여부를 결과 해석 시 별도 언급

### 외국인 비율 계산
```python
df['외국인비율'] = df['등록외국인현황'] / df['주민등록인구'] * 100
```

### ⚠️ 범죄율(Y) 정의 통일
```
모든 변수를 "인구 대비 비율" 또는 "1인당 지표"로 통일해야 함
✔ 범죄율      → 인구 10만명당 범죄 발생 건수
✔ 실업률      → % (비율) ✅
✔ 음주율      → % (비율) ✅
✔ 물가 상승률 → % (비율) ✅
✔ 인구 수     → 명 (절대값) → 그대로 사용
✔ 외국인      → 등록외국인수 ÷ 인구수로 비율 변환 필요!
✔ 기초생활수급자수 → 인구수 대비 비율로 변환 권장
```
